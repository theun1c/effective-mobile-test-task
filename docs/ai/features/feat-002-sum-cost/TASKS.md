# TASKS.md

## Фича

Подсчёт суммарной стоимости подписок.

## Задачи

- [ ] Создать DTO для query-параметров.
- [ ] Добавить `from`.
- [ ] Добавить `to`.
- [ ] Добавить опциональный `user_id`.
- [ ] Добавить опциональный `service_name`.
- [ ] Создать DTO ответа с полем `total_cost`.

### 2. Валидация

- [ ] Проверять наличие `from`.
- [ ] Проверять наличие `to`.

- [ ] Проверять, что `to >= from`.
- [ ] Проверять `user_id`, если он передан.
- [ ] Проверять `service_name`, если он передан.

### 3. Алгоритм расчёта

- [ ] Сложить стоимость всех подходящих подписок.

### 4. Repository
